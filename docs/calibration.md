# Calibration

> Stub — calibration ships with v0.2 (milestone M4). See SPEC.md §4.

Planned procedure: button-triggered 2-second rest on the stadium floor (or a
rim jig) zeroes the IMU orientation to the stadium plane; the offset is
stored in NVS. Launch angle is then reported relative to that plane.

Nothing to calibrate in v0.1 — spin (RPM) measurement needs no calibration.
