# M1 Housing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Produce a printable, non-destructive two-part shell that mounts on the winder launcher's top face and houses the M0b-validated sensor module, presence microswitch, XIAO ESP32-C3, LiPo battery, and power switch — modeled in Blender via an MCP connection against a real 3D scan of the launcher.

**Architecture:** A phone scan of the launcher's top face becomes a fixed reference mesh in Blender; Claude scripts the shell geometry (via `bpy`, driven through a Blender MCP server) as boolean operations against that reference and against a set of measured component dimensions. Finished parts export to STL, get converted to STEP via a small headless FreeCAD script, and get physically print-tested in a loop against the acceptance criteria from the design spec.

**Tech Stack:** Blender (with a Blender MCP addon/server), Python (`bpy` inside Blender; `trimesh` for mesh verification outside it), FreeCAD (headless, scripted, STEP conversion only), PlatformIO firmware already in place (unaffected by this plan).

## Global Constraints

- Zero launcher modification (hard requirement, `SPEC.md` §1): the mount must be a removable, non-destructive friction-fit/clip. Never glue, drill, or permanently press anything into the launcher.
- Print goal: PLA, no supports (`SPEC.md` §6). Design overhangs/split-lines accordingly.
- Deliverables per `SPEC.md` §6: STEP + STL + source. Source format is `.blend` (not `.f3d` — updated by this plan, see Task 1).
- Repo layout (design spec): raw scans in `models/scans/` (gitignored, never committed); finished deliverables in `models/housing/`.
- Sensor: the breakout module already wired and validated at M0b (VCC/GND/AO/DO, onboard LM393 comparator) — not the bare TCRT5000 from the original BOM.
- Design spec of record: `docs/superpowers/specs/2026-07-09-housing-design.md`. This plan implements it; if anything here conflicts with it, the spec wins and this plan should be corrected.

- [ ] **Step 3: Write the scan verification script**

Create `models/housing/verify_scan.py`:

```python
"""
Sanity-checks the raw scan mesh before any modeling work starts.
Run with: python3 models/housing/verify_scan.py
Requires: pip install trimesh
"""
import sys
import trimesh

SCAN_PATH = "models/scans/launcher-top-face.stl"

def main():
    try:
        mesh = trimesh.load(SCAN_PATH)
    except Exception as e:
        print(f"FAIL: could not load {SCAN_PATH}: {e}")
        sys.exit(1)

    print(f"Loaded {SCAN_PATH}")
    print(f"  vertices: {len(mesh.vertices)}")
    print(f"  faces:    {len(mesh.faces)}")
    print(f"  bounding box (mm): {mesh.bounding_box.extents}")
    print(f"  watertight: {mesh.is_watertight}")

    extents = mesh.bounding_box.extents
    # A real launcher top-face scan should span tens of millimeters, not
    # meters (unit mismatch) or fractions of a millimeter (degenerate scan).
    if not all(5.0 <= e <= 300.0 for e in extents):
        print("FAIL: bounding box outside 5-300mm range on some axis — "
              "check the scan's export units (should be mm).")
        sys.exit(1)

    print("PASS: scan mesh loads and dimensions look plausible.")

if __name__ == "__main__":
    main()
```

- [ ] **Step 1: Write the modeling script**

Create `models/housing/blender_scripts/02_sensor_carrier.py`:

```python
import bpy
import sys
sys.path.insert(0, "/Users/neilmedici/Documents/GitHub/OpenRip/models/housing")
import measurements as m

MM = 0.001  # bpy works in meters; every *_mm value below is converted via mm()
CLEARANCE_MM = 0.3      # per-side fit clearance for a snug (not press) pocket fit
FLOOR_THICKNESS_MM = 1.5  # solid material left under the pocket, and the walls' target thickness

def mm(v):
    return v * MM

# Sensor module pocket: a rectangular recess sized to the module's PCB
# plus clearance, deep enough for the PCB thickness plus dome protrusion,
# with FLOOR_THICKNESS_MM of solid material left below it. The block's
# own height is the pocket depth plus that floor — NOT the same number.
pocket_l_mm = m.MODULE_PCB_LENGTH_MM + 2 * CLEARANCE_MM
pocket_w_mm = m.MODULE_PCB_WIDTH_MM + 2 * CLEARANCE_MM
pocket_depth_mm = m.MODULE_PCB_THICKNESS_MM + m.MODULE_DOME_PROTRUSION_MM + 0.5
block_l_mm = pocket_l_mm + 4.0   # 2mm wall each side
block_w_mm = pocket_w_mm + 4.0
block_h_mm = pocket_depth_mm + FLOOR_THICKNESS_MM

window = bpy.data.objects["WindowCenter"]
plunger = bpy.data.objects["PlungerCenter"]

# Outer block. primitive_cube_add(size=1) makes a 1m-per-side cube, so
# object.scale == the final dimensions in meters directly (no extra factor).
bpy.ops.mesh.primitive_cube_add(
    size=1, location=(window.location.x, window.location.y, mm(block_h_mm) / 2)
)
carrier = bpy.context.object
carrier.name = "SensorCarrier"
carrier.scale = (mm(block_l_mm), mm(block_w_mm), mm(block_h_mm))
bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)

# Cut the module pocket down from the top, leaving FLOOR_THICKNESS_MM intact.
cutter_z = mm(block_h_mm) - mm(pocket_depth_mm) / 2
bpy.ops.mesh.primitive_cube_add(
    size=1, location=(window.location.x, window.location.y, cutter_z)
)
cutter = bpy.context.object
cutter.scale = (mm(pocket_l_mm), mm(pocket_w_mm), mm(pocket_depth_mm) + mm(1.0))
bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)

mod = carrier.modifiers.new(name="PocketCut", type='BOOLEAN')
mod.operation = 'DIFFERENCE'
mod.object = cutter
bpy.context.view_layer.objects.active = carrier
bpy.ops.object.modifier_apply(modifier="PocketCut")
bpy.data.objects.remove(cutter, do_unlink=True)

# Switch carriage slot: a through-slot near the plunger, long enough to
# slide the microswitch body toward/away from it.
slot_travel_mm = 4.0  # adjustable range, per docs/build-guide.md carriage note
slot_l_mm = m.MICROSWITCH_BODY_LENGTH_MM + slot_travel_mm
slot_w_mm = m.MICROSWITCH_BODY_WIDTH_MM + 2 * CLEARANCE_MM

bpy.ops.mesh.primitive_cube_add(
    size=1, location=(plunger.location.x, plunger.location.y, mm(block_h_mm) / 2)
)
slot_cutter = bpy.context.object
slot_cutter.scale = (mm(slot_l_mm), mm(slot_w_mm), mm(block_h_mm) + mm(2.0))
bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)

mod2 = carrier.modifiers.new(name="SlotCut", type='BOOLEAN')
mod2.operation = 'DIFFERENCE'
mod2.object = slot_cutter
bpy.context.view_layer.objects.active = carrier
bpy.ops.object.modifier_apply(modifier="SlotCut")
bpy.data.objects.remove(slot_cutter, do_unlink=True)

bpy.ops.wm.save_mainfile()
print("SensorCarrier bounding box (m):", carrier.dimensions)
```

- [ ] **Step 3: Write the dimension verification script**

Create `models/housing/verify_part.py`:

```python
"""
Checks an exported housing part's bounding box against measurements.py.
Run with: python3 models/housing/verify_part.py sensor_carrier
Requires: pip install trimesh
"""
import sys
import trimesh
import measurements as m

PARTS = {
    "sensor_carrier": {
        "path": "models/housing/exports/sensor_carrier.stl",
        # (min_l, min_w, min_h) the part must be at least this big to
        # actually contain its pocket contents.
        "min_extents_mm": (
            m.MODULE_PCB_LENGTH_MM,
            m.MODULE_PCB_WIDTH_MM,
            m.MODULE_PCB_THICKNESS_MM,
        ),
    },
}

def main():
    if len(sys.argv) != 2 or sys.argv[1] not in PARTS:
        print(f"Usage: python3 verify_part.py [{'|'.join(PARTS)}]")
        sys.exit(1)

    spec = PARTS[sys.argv[1]]
    mesh = trimesh.load(spec["path"])
    if not mesh.is_watertight:
        print(f"FAIL: {spec['path']} is not watertight (not print-ready).")
        sys.exit(1)

    extents = sorted(mesh.bounding_box.extents)
    minimums = sorted(spec["min_extents_mm"])
    if any(e < mn for e, mn in zip(extents, minimums)):
        print(f"FAIL: {spec['path']} bounding box {extents} smaller than "
              f"required minimums {minimums}.")
        sys.exit(1)

    print(f"PASS: {spec['path']} is watertight and at least {minimums} mm.")

if __name__ == "__main__":
    main()
```

```python
import bpy
import sys
sys.path.insert(0, "/Users/neilmedici/Documents/GitHub/OpenRip/models/housing")
import measurements as m

MM = 0.001
CLEARANCE_MM = 0.3
FLOOR_THICKNESS_MM = 1.5
WALL_MM = 2.0

def mm(v):
    return v * MM

def add_pocket(carrier, name, center_x_mm, center_y_mm, l_mm, w_mm, depth_mm, block_h_mm):
    """Cuts a rectangular pocket into `carrier`, centered at (center_x_mm,
    center_y_mm) in world mm, recessed depth_mm from the block's top."""
    cutter_z = mm(block_h_mm) - mm(depth_mm) / 2
    bpy.ops.mesh.primitive_cube_add(size=1, location=(mm(center_x_mm), mm(center_y_mm), cutter_z))
    cutter = bpy.context.object
    cutter.scale = (mm(l_mm), mm(w_mm), mm(depth_mm) + mm(1.0))
    bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)
    mod = carrier.modifiers.new(name=name, type='BOOLEAN')
    mod.operation = 'DIFFERENCE'
    mod.object = cutter
    bpy.context.view_layer.objects.active = carrier
    bpy.ops.object.modifier_apply(modifier=name)
    bpy.data.objects.remove(cutter, do_unlink=True)

# Three pockets side by side along the bay's length: XIAO, battery, switch.
xiao_l = m.XIAO_LENGTH_MM + 2 * CLEARANCE_MM
xiao_w = m.XIAO_WIDTH_MM + 2 * CLEARANCE_MM
xiao_d = m.XIAO_THICKNESS_MM + 0.5

batt_l = m.BATTERY_LENGTH_MM + 2 * CLEARANCE_MM
batt_w = m.BATTERY_WIDTH_MM + 2 * CLEARANCE_MM
batt_d = m.BATTERY_THICKNESS_MM + 0.5

sw_l = m.SWITCH_LENGTH_MM + 2 * CLEARANCE_MM
sw_w = m.SWITCH_WIDTH_MM + 2 * CLEARANCE_MM
sw_d = m.SWITCH_HEIGHT_MM + 0.5

# Lay the three pockets out left-to-right with WALL_MM between each.
bay_w_mm = max(xiao_w, batt_w, sw_w) + 2 * WALL_MM
bay_l_mm = xiao_l + batt_l + sw_l + 4 * WALL_MM
deepest_pocket_mm = max(xiao_d, batt_d, sw_d)
bay_h_mm = deepest_pocket_mm + FLOOR_THICKNESS_MM

x0 = -bay_l_mm / 2 + WALL_MM + xiao_l / 2
x1 = x0 + xiao_l / 2 + WALL_MM + batt_l / 2
x2 = x1 + batt_l / 2 + WALL_MM + sw_l / 2

bpy.ops.mesh.primitive_cube_add(size=1, location=(0, 0, mm(bay_h_mm) / 2))
bay = bpy.context.object
bay.name = "ElectronicsBay"
bay.scale = (mm(bay_l_mm), mm(bay_w_mm), mm(bay_h_mm))
bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)

add_pocket(bay, "XiaoPocket", x0, 0, xiao_l, xiao_w, xiao_d, bay_h_mm)
add_pocket(bay, "BatteryPocket", x1, 0, batt_l, batt_w, batt_d, bay_h_mm)
add_pocket(bay, "SwitchPocket", x2, 0, sw_l, sw_w, sw_d, bay_h_mm)

# USB-C cutout: through the wall at the XIAO pocket's connector edge (the
# -Y wall, assuming the XIAO's USB-C port faces -Y in its pocket).
usbc_x = x0
usbc_y = -bay_w_mm / 2
bpy.ops.mesh.primitive_cube_add(size=1, location=(mm(usbc_x), mm(usbc_y), mm(bay_h_mm) - mm(xiao_d) / 2))
usbc_cutter = bpy.context.object
usbc_cutter.scale = (mm(8.0), mm(WALL_MM) + mm(1.0), mm(3.0))
bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)
mod = bay.modifiers.new(name="UsbcCut", type='BOOLEAN')
mod.operation = 'DIFFERENCE'
mod.object = usbc_cutter
bpy.context.view_layer.objects.active = bay
bpy.ops.object.modifier_apply(modifier="UsbcCut")
bpy.data.objects.remove(usbc_cutter, do_unlink=True)

# Slide-switch actuator slot: through the top face, above the switch pocket.
bpy.ops.mesh.primitive_cube_add(size=1, location=(mm(x2), 0, mm(bay_h_mm)))
sw_cutter = bpy.context.object
sw_cutter.scale = (mm(m.SWITCH_LENGTH_MM), mm(2.0), mm(2.0))
bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)
mod = bay.modifiers.new(name="SwitchSlotCut", type='BOOLEAN')
mod.operation = 'DIFFERENCE'
mod.object = sw_cutter
bpy.context.view_layer.objects.active = bay
bpy.ops.object.modifier_apply(modifier="SwitchSlotCut")
bpy.data.objects.remove(sw_cutter, do_unlink=True)

bpy.ops.wm.save_mainfile()
print("ElectronicsBay bounding box (m):", bay.dimensions)
```

Create `models/housing/blender_scripts/04_mounting_and_joints.py`. This is the one genuinely scan-dependent step the design spec flagged as an open question (exact clip geometry). Implement it as: intersect a thin (2mm) shell offset from `LauncherScan`'s top-face surface directly beneath `SensorCarrier`'s footprint, using Blender's Shrinkwrap modifier to conform a flat plate to the scan surface, then boolean-UNION that conformed plate onto `SensorCarrier`'s underside so it registers against the real contour instead of a flat assumption:

```python
import bpy

carrier = bpy.data.objects["SensorCarrier"]
scan = bpy.data.objects["LauncherScan"]

bpy.ops.mesh.primitive_plane_add(size=0.03, location=carrier.location)
plate = bpy.context.object
plate.name = "RegistrationPlate"
solidify = plate.modifiers.new(name="Solidify", type='SOLIDIFY')
solidify.thickness = 0.002
shrinkwrap = plate.modifiers.new(name="Shrinkwrap", type='SHRINKWRAP')
shrinkwrap.target = scan
shrinkwrap.wrap_method = 'NEAREST_SURFACEPOINT'
bpy.context.view_layer.objects.active = plate
bpy.ops.object.modifier_apply(modifier="Solidify")
bpy.ops.object.modifier_apply(modifier="Shrinkwrap")

union = carrier.modifiers.new(name="RegistrationUnion", type='BOOLEAN')
union.operation = 'UNION'
union.object = plate
bpy.context.view_layer.objects.active = carrier
bpy.ops.object.modifier_apply(modifier="RegistrationUnion")
bpy.data.objects.remove(plate, do_unlink=True)

print("Registration plate conformed and unioned into SensorCarrier.")
bpy.ops.wm.save_mainfile()
```

Snap-fit tabs joining `SensorCarrier` to `ElectronicsBay`: run this immediately after the registration-plate code above, in the same script. It reads both objects' actual `.dimensions` live rather than requiring anyone to copy numbers by hand:

```python
bay = bpy.data.objects["ElectronicsBay"]
carrier = bpy.data.objects["SensorCarrier"]  # already updated above

TAB_WIDTH_M = 0.003
TAB_DEPTH_M = 0.0015

# Tabs are added to ElectronicsBay's face closest to SensorCarrier. This
# assumes the bay sits at +X relative to the carrier — check the Step 3
# viewport screenshot; if they're arranged along Y instead, swap the X
# references below for Y ones before running.
bay_min_x = bay.location.x - bay.dimensions.x / 2
tab_y_positions = [
    bay.location.y - bay.dimensions.y / 4,
    bay.location.y + bay.dimensions.y / 4,
]

for i, y in enumerate(tab_y_positions):
    bpy.ops.mesh.primitive_cube_add(
        size=1, location=(bay_min_x - TAB_DEPTH_M / 2, y, bay.location.z)
    )
    tab = bpy.context.object
    tab.scale = (TAB_DEPTH_M, TAB_WIDTH_M, TAB_WIDTH_M)
    bpy.ops.object.transform_apply(location=False, rotation=False, scale=True)
    union = bay.modifiers.new(name=f"TabUnion_{i}", type='BOOLEAN')
    union.operation = 'UNION'
    union.object = tab
    bpy.context.view_layer.objects.active = bay
    bpy.ops.object.modifier_apply(modifier=f"TabUnion_{i}")
    bpy.data.objects.remove(tab, do_unlink=True)

# Matching slots cut into SensorCarrier at the same Y positions, sized
# 20% larger than the tabs for a snap-fit (not press-fit) clearance.
carrier_max_x = carrier.location.x + carrier.dimensions.x / 2
for i, y in enumerate(tab_y_positions):
    bpy.ops.mesh.primitive_cube_add(
        size=1, location=(carrier_max_x + TAB_DEPTH_M / 2, y, bay.location.z)

    slot = bpy.context.object
    slot.scale = (TAB_DEPTH_M * 1.2, TAB_WIDTH_M * 1.2, TAB_WIDTH_M * 1.2)

    diff.operation = 'DIFFERENCE'
    diff.object = slot
    bpy.context.view_layer.objects.active = carrier
    bpy.ops.object.modifier_apply(modifier=f"TabSlot_{i}")
    bpy.data.objects.remove(slot, do_unlink=True)

bpy.ops.wm.save_mainfile()
print("Snap tabs added:", [t for t in bay.modifiers.keys() if t.startswith("TabUnion")])
```

```python
"""
Converts housing STL exports to STEP using FreeCAD's headless Python API.
Run with: freecad --console models/housing/freecad_convert.py
(or the full path to FreeCAD's bundled Python if `freecad` isn't on PATH)
"""
import FreeCAD
import Part
import Mesh
import os

PARTS = ["sensor_carrier", "electronics_bay"]
BASE = "/Users/neilmedici/Documents/GitHub/OpenRip/models/housing/exports"

for part in PARTS:
    stl_path = os.path.join(BASE, f"{part}.stl")
    step_path = os.path.join(BASE, f"{part}.step")

    doc = FreeCAD.newDocument(part)
    mesh_obj = doc.addObject("Mesh::Feature", "Mesh")
    mesh_obj.Mesh = Mesh.Mesh(stl_path)

    shape = Part.Shape()
    shape.makeShapeFromMesh(mesh_obj.Mesh.Topology, 0.05)  # 0.05mm sewing tolerance
    solid = Part.makeSolid(shape) if shape.Faces else shape

    part_obj = doc.addObject("Part::Feature", "Solid")
    part_obj.Shape = solid
    doc.recompute()

    Part.export([part_obj], step_path)
    print(f"Converted {stl_path} -> {step_path}")
    FreeCAD.closeDocument(doc.Name)
```
